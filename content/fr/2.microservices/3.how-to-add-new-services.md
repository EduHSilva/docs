Para adicionar novos serviços é necessario registra-los no service-discover

## Spring
::steps{level="3"}
### Dependencia do eureka
```
<dependency>
    <groupId>org.springframework.cloud</groupId>
	<artifactId>spring-cloud-starter-netflix-eureka-client</artifactId>
    <scope>provided</scope>
</dependency>
```
### .properties
```
eureka.client.service-url.defaultZone=urlDoServiceDiscover
eureka.client.fetch-registry=true
eureka.client.register-with-eureka=true
```
::
Seu novo serviço deve estar disponivel em: http://localhost:8080/nomeDoServico
