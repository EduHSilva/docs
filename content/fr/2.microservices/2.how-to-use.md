### Compra
Na compra de um microservice isolado, o gateway e service-discover já vão incluidos,
posibilitando que o usuário crie e plugue mais micro-serviços.

#### GitHub
Se adquirido todos os microservices, há a possibilidade de clonar via git:
```code
git clone https://github.com/EduHSilva/docs.git
```
#### ZIP
Se adquirido somente um ou microservices especificos, será fornecido um .zip, sendo necessario extrai-lo.

### Requisitos
- Docker
- Java

### Variaveis de ambiente
As variaveis de ambiente devem ser configuradas em cada serviço adquirido.
Cada serviço terá seu applications-example.properties, sendo necessario criar um equilavente para o docker
```
appplication-docker.properties
```
e no application.properties
```
spring.profiles.active=docker
```

### Como executar
Com os requisitos instalados e as variaveis de ambiente configurados corretamente:
```
docker compose up
```
